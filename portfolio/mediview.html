Med-I-View App

This project was to create a mobile app of your choosing.

My concept was a medical app that allows parents easy access to their children's medical records from their phones. It also allows printing access without having to visit the actual doctor's office. It is secure through face id and password login.

This image is of my high-fidelity, interactive prototype made using Figma.


The other part of this project was to make your own logo for the app you create. I drew this logo myself using Adobe Illustrator.
