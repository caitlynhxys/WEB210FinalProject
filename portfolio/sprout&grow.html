Sprout and Grow Website

This website was created as a part of my Graphic Design course at Wake Technical Community College.

I designed this site from scratch using VS Code, with all images, text, and colors chosen by me.

The assignment was to create a website for a local Raleigh Plant Nursery.

This project required creating a 7 page layout, incluing a navigation bar on all pages.

One of the pages I included was a Plants page to showcase the nursery's inventory.
